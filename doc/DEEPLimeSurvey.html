submitChoice()

Handler for the Submit button that is clicked after a user selects a choice. Checks if a choice has been selected (if not, shows an error), then calls DEEPCore.saveChoice() to save the selected choice. Then, to display the next choice, it clears the next choice and gets the next question from DEEPCore. DEEPCore can return a set of choices, which then get passed to updateChoices(), or it can also return false, which means that DEEP is over, after which DEEPLimeSurvey will call finish() to complete the DEEP process.
